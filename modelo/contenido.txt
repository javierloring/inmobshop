En esta sección se crean las clases que permiten interactuar con la base de
datos de forma genérica y con las entidades de la aplicación en particular:
- Clase BD
    - función conectar: conecta a la base de datos y aplica el modo de errores
    PDOException.

    - función obtener_registros: obtener los registros de la tabla de la base de
     datos conectada.

    - función numero_registros: devuelve el nº de registros de la tabla de la BD

    - función seleccionar_ids_tabla: obtiene los ids de una tabla. (convertir
    desde producto).

    - función obten_registro_id(id, tabla, dbh):obtener el registro dado un id
    una tabla y una base de datos (una conexión).

    - función borra_registro_id(id, tabla, dbh): borrar un registro dado un id
    una tabla y una base de datos (una conexión).

    - función inserta_registro_id(tabla, dbh, campos, valores): inserta un
    registro en una tabla de una base de datos conectada pasando los campos a
    insertar y los valores para los mismos.

    - función actualiza_registro(id, tabla, dbh, campos, valores): actualizar un
    registro de una tabla de una base de datos conectada, conocido el id, el
    nombre de la tabla y los campos a actualizar y los nuevos valores.
